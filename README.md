# E-commerce Project (Scaffold)

This repository contains the initial project structure for an e-commerce application.

Technology stack
- Frontend: React + Vite, Tailwind CSS, React Router
- Backend: FastAPI (separate folder)
- Database: MySQL

Project layout

- frontend/: React + Vite app
- backend/: FastAPI app skeleton

Quick start

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend (basic)

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
# run uvicorn when ready
```

Notes
- This is only the initial scaffold. No backend business logic or AI integrations added yet.
